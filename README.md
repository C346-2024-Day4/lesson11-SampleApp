# Hello this is my app

The app will interact with a RESTful web service created using https://retool.com/api-generator

https://retoolapi.dev/3mPAvR/contact

Endpoints:

GET    /contact
GET    /contact/:id
POST   /contact
PUT    /contact/:id
PATCH  /contact/:id
DELETE /contact/:id
